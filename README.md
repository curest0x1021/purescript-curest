- [Module Documentation](docs/Text/Handlebars.md)
- [Example](test/Main.purs)

## Usage

To build the test project, clone this repo and then run:

    pulp dep update
    pulp build
    pulp test

To use this library in your own project:

    pulp dep i purescript-handlebars
